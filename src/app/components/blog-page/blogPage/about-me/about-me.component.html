About Me

Who am I?

I am a 19 year old from Ireland with many interests from sport to reading and technology.

I did my Leaving Certificate last year (the Irish State exams for leaving secondary/high school). I took a gap year this academic year, and have spent the year working, learning and thinking about what I want to do in the future. I have always been interested in technology, and this interest of mine has really blossomed in the past few months, and I have learned many things from web development to Machine Learning and Artificial Intelligence.

What got me into technology?

During one of the COVID lockdowns I started to experiment with Shopify and Drop Shipping (selling dog beds and neck massagers).

While doing this, I began teaching myself some new skills in coding, 3D Rendering with Blender, video graphics with the Adobe Suite, and any skills I could get my hands on that would help me to build better user experiences on Shopify.

Software Technologies

The following is a list of the software technologies that I have been learning over the last while:

  • React: A component based Javascript/Typescript framework to build modern single-page web applications.
    • GSAP: A Javascript framework (with React hooks) for building animations in the browser. I gravitated towards this framework because it's timeline features are similar to keyframe oriented animation in Adobe products like After Effects and Premiere Pro.
    • Framer Motion: Another Javascript framework for building animations. I no longer use this framework and have completely converted to GSAP because GSAP has a more inuitive and flexible development experience.
  • Tensorflow: A Python machine learning and artificial intelligence library which is useful for building and training Neural Networks of all kinds
    • Keras: A Tensorflow library for creating Neural Networks for Deep Learning. Keras makes it even more simple to create Neural Networks. It has many out-of-the-box features for creating both simple and complex models, and useful uilities for preprocessing data.
    • OpenCV: Another AI library, however this library is a Computer Vision specific (a subfield of Artificial Intelligence that aims at making computers able to see). OpenCV makes it easier to create Computer Vision models of all kinds.
  • Continuous Integration / Continuous Delivery (CI/CD): I have used GitHub Actions for automating the delivery of some of my projects.
  • Docker: A containerization technology that allows developers to create ( and most importantly to recreate ) lightweight virtualized environments at the operating system level, either locally or remotely. Docker is something I have only experimented with ( and used in indirectly via GitHub Actions ) so far and look forward to learning in more detail.
